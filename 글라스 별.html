칵테일을 담는 글라스별로 레시피 소개

칵테일 글라스

  1. 바카디 : 칵테일 글라스에 얼음 - 셰이커에 얼음, 화이트럼 1+3/4oz, 라임주스 3/4oz, 그레나딘 시럽 1tsp - 셰이킹 - 칵테일 글라스에 얼음 버리고 음료 따르기.
  2. 다이키리 : 칵테일 글라스에 얼음 - 셰이커에 얼음, 화이트럼 1+3/4oz, 라임주스 3/4oz, 설탕 1tsp - 셰이킹 - 글라스의 얼음 버리고 음료 따르기.
  3. 드라이 마티니 : 칵테일 글라스에 얼음 - 믹싱글라스에 얼음, 드라이진 2oz, 드라이베르무트 1/3oz - 스터 - 글라스에 얼음 버리고 음료 따르기 - 칵테일 픽에 올리브로 장식.
  4. 애플 마티니 : 칵테일 글라스에 얼음 - 셰이커에 얼음, 보드카 1oz, 애플퍼커 1oz, 라임주스 1/2oz - 셰이킹 - 글라스에 얼음 버리고 음료 따르기 - 사과슬라이스로 장식.
  5. 코스모폴리탄 : 칵테일 글라스에 얼음 - 셰이커에 얼음, 보드카 1oz, 트리플섹 1/2oz, 라임주스 1/2oz, 크랜베리주스 1/2oz - 셰이킹 - 글라스에 얼음 버리고 음료 따르기 - 라임 또는 레몬의 껍질을 꼬아 장식.
  6. 마가리타 : 칵테일 글라스에 얼음 - 셰이커에 얼음, 테킬라 1+1/2oz, 트리플섹 1/2oz, 라임주스 1/2oz - 셰이킹 - 글라스에 얼음 버리고, 물기 닦기 - 글라스 림에 레몬즙 바르고 소금 묻히기 - 음료 따르기.
  7. 맨해튼 : 칵테일 글라스에 얼음 - 믹싱글라스에 얼음, 버번위스키 1+1/2oz, 스윗베르무트 3/4oz, 앙고스투라비터 1dash - 스터 - 글라스에 얼음 버리고 음료 따르기 - 칵테일 픽을 사용한 체리 장식.
  8. 뉴욕 : 칵테일 글라스에 얼음 - 셰이커에 얼음, 버번위스키 1+1/2oz, 라임주스 1/2oz, 설탕 1tsp, 그레나딘시럽 1/2tsp - 셰이킹 - 글라스에 얼음 버리고 음료 따르기 - 레몬 껍질 꼬아 장식.
  9. 브랜디 알렉산더 : 칵테일 글라스에 얼음 - 셰이커에 얼음, 브랜디 3/4oz, 크렘드카카오 3/4oz, 우유 3/4oz - 셰이킹 - 글라스에 얼음 버리고 음료 따르 - 너트메그가루 뿌려 장식.
  10. 허니문 : 칵테일 글라스에 얼음 - 셰이커에 얼음, 애플브랜디 3/4oz, 베네딕틴 3/4oz, 트리플섹 1/4oz, 레몬주스 1/2oz - 글라스에 얼음 버리고 음료 따르기.
  11. 사이드카 : 칵테일 글라스에 얼음 - 셰이커에 얼음, 브랜디 1oz, 쿠앵트로 또는 트리플섹 1oz, 레몬주스 1/4oz - 셰이킹 - 글라스에 얼음 버리고 음료 따르기.
  12. 애프리코트 : 칵테일 글라스에 얼음 - 셰이커에 얼음, 애프리코트브랜디 1+1/2oz, 드라이진 1tsp, 레몬주스 1/2oz, 오랜지주스 1/2oz - 셰이킹 - 글라스에 얼음 버리고 음료 따르기.
  13. 금산 : 칵테일 글라스에 얼음 - 셰이커에 얼음, 금산인삼주(43도) 1+1/2oz, 커피리큐르 1/2oz, 애플퍼커1/2oz, 라임주스 1tsp - 셰이킹 - 글라스에 얼음 버리고 음료따르기.
  14. 힐링 : 칵테일 글라스에 얼음 - 셰이커에 얼음, 감홍로(40도) 1+1/2oz, 베네딕틴 1/3oz, 카시스 1/3oz, 스윗앤사워믹스 1oz - 셰이킹 - 글라스에 얼음 버리고 음료 따르기 - 레몬껍질 꼬아서 장식.
  15. 진도 : 칵테일 글라스에 얼음 - 셰이커에 얼음, 진도홍주(40도) 1oz, 크렘드민(하얀색) 1/2oz, 청포도주스 3/4oz, 라즈베리시럽 1/2oz - 셰이킹 - 글라스에 얼음 버리고 음료 따르기.
  16. 풋사랑 : 칵테일 글라스에 얼음 - 셰이커에 얼음, 안동소주(35도) 1oz, 트리플섹 1/3oz, 애플퍼커 1oz, 라임주스 1/3oz - 셰이킹 - 글라스에 얼음 버리고 음료 따르기 - 사과 슬라이스로 장식.

하이볼 글라스

  1. 프레시 레몬 스쿼시 : 하이볼 글라스에 레몬 1/2개 스퀴징 - 설탕 2tsp - 스터 - 글라스에 얼음 - 글라스의 80%까지 탄산수, 2~3회 스터 - 레몬 슬라이스로 장식.
  2. 진 피즈 : 하이볼 글라스에 얼음 - 셰이커에 얼음, 진 1+1/2oz, 레몬주스 1/2oz, 설탕 1tsp - 셰이킹 - 글라스에 음료 따르기 - 글라스의 80%까지 탄산수 채우고 2~3회 젓기 - 레몬 슬라이스로 장식.
  3. 쿠바 리브레 : 하이볼 글라스에 얼음, 화이트럼 1+1/2oz, 라임주스 1/2oz - 글라스의 80%를 콜라로 채우고 2~3번 젓기 - 레몬조각으로 장식.
  4. 모스코 뮬 : 하이볼 글라스에 얼음, 보드카 1+1/2oz, 라임 주스 1/2oz, 글라스에 80%까지 진저에일 채우고 2~3회 스터 - 라임 또는 레몬 슬라이스로 장식.
  5. 시브리즈 : 하이볼 글라스에 얼음, 보드카 1+1/2oz, 크랜베리주스 3oz, 자몽주스 1/2oz - 스터 - 레몬 또는 라임 조각으로 장식.

필스너 글라스

  1. 블루 하와이안 : 필스너 글라스에 얼음 - 블렌더에 얼음, 화이트럼 1oz, 블루큐라소 1oz, 코코넛럼 1oz, 파인애플주스 2+1/2oz - 블렌딩 - 글라스에 얼음 버리고 음료 따르기 - 파인애플조각과 체리로 장식.
  2. 마이 타이 : 필스너 글라스에 얼음 - 블렌더에 얼음, 화이트럼 1+1/4oz, 트리플섹 3/4oz, 라임주스 1oz, 파인애플주스 1oz, 오렌지주스 1oz, 그레나딘시 1/4oz - 블렌딩 - 글라스에 얼음 버리고 음료 따르기 - 파인애플조각과 체리로 장식.
  3. 피냐 콜라다 : 필스너 글라스에 얼음 - 블렌더에 얼음, 화이트럼 1+1/4oz, 피나콜라다믹스 2oz, 파인애플주스 2oz - 블렌딩 - 글라스에 얼음 버리고 음료 따르기 - 파인애플조각과 체리로 장식.
  4. 싱가포르 슬링 : 필스너 글라스에 얼음 - 셰이커에 얼음, 드라이진 1+1/2oz, 레몬주스 1/2oz, 설탕 1tsp - 셰이킹 - 글라스에 음료 따르기 - 글라스에 80%까지 탄산수 채우고 2~3회 젓기 - 체리브랜디 1/2oz 띄우고, 오렌지 슬라이스와 체리로 장식.
  5. 테킬라 선라이즈 : 필스너 글라스에 얼음, 테킬라 1+1/2oz - 글라스의 80%까지 오랜지주스 채우기 - 스터 - 그레나딘시럽 1/2oz를 바스푼을 이용해 중간에 넣기.
  6. 버진 프루트 펀치 : 필스너 글라스에 얼음 - 블렌더에 얼음, 오렌지주스 1oz, 파인애플주스 1oz, 크랜베리주스 1oz, 자몽주스 1oz, 레몬주스 1/2oz, 그레나딘 시럽 1/2oz - 블렌딩 - 글라스에 얼음 버리고 음료 따르기 - 파인애플조각과 체리로 장식.

올드 패션드 글라스

  1. 네그로니 : 올드 패션드 글라스에 얼음, 드라이진 3/4oz, 스윗베르무트 3/4oz, 캄파리 3/4oz - 스터 - 레몬껍질 꼬아서 장식.
  2. 블랙 러시안 : 올드 패션드 글라스에 얼음 - 보드카 1oz, 커피리큐르 1/2oz - 스터.
  3. 올드 패션드 : 올드 패션드 글라스에 설탕 1tsp, 앙고스투라비터 1dash, 탄산수 1/2oz - 스터 - 글라스에 얼음 - 버번위스키 1+1/2oz - 스터 - 오렌지 슬라이스와 체리로 장식.
  4. 러스티 네일 : 올드 패션드 잔에 얼음, 스카치위스키 1oz, 드람부이 1/2oz - 스터.

콜린스 글라스

  1. 롱아일랜드 아이스티 : 콜린스 글라스에 얼음, 진 1/2oz, 보드카 1/2oz, 화이트럼 1/2oz, 테킬라1/2oz, 트리플섹1/2oz, 스윗앤사워믹스 1+1/2oz - 글라스에 80%까지 콜라 채우고 스터 - 라임 또는 레몬 조각으로 장식.
  2. 준 벅 : 콜린스 글라스에 얼음 - 셰이커에 얼음, 미도리 1oz, 코코넛럼 1/2oz, 바나나리큐르 1/2oz, 파인애플주스 2oz, 스윗앤사워믹스 2oz - 셰이킹 - 음료 따르고 파인애플조각과 체리로 장식.

소서형 샴페인 글라스

  1. 그래스 호퍼 : 소서형 글라스에 얼음 - 셰이커에 얼음, 크렘드민트(초록색) 1oz, 크렘드카카오(하얀색) 1oz, 우유 1oz - 셰이킹 - 글라스에 얼음 버리고 음료 따르기.

플루트형 샴페인 글라스

  1. 고창 : 플루트 글라스에 얼음 - 믹싱 글라스에 얼음, 선운산복분자주 2칵테일 글라스에 얼음 - 셰이커에 얼음, 감홍로(40도) 1+1/2oz, 베네딕틴 1/3oz, 카시스 1/3oz, 스윗앤사워믹스 1oz - 셰이킹 - 글라스에 얼음 버리고 음료 따르기 - 레몬껍질 꼬아서 장식., 쿠앵트로 또는 트리플섹 1/2oz, 사이다 2oz - 스터 - 글라스에 얼음 버리고 음료 따르기.

셰리 글라스

  1. B-52 : 셰리 글라스에 바스푼으로 띄우기 - 커피리큐르 1/3part, 베일리스 1/3part, 그랑마니에 1/3part.

사워 글라스

  1. 위스키 사워 : 사워 글라스에 얼음 - 셰이커에 얼음, 버번위스키 1+1/2oz, 레몬주스 1/2oz, 설탕 1tsp - 셰이킹 - 글라스에 얼음 버리고 음료 따르기 - 탄산수 넣고 2~3회 스터 - 레몬 슬라이스와 체리로 장식.

코디얼 글라스

  1. 푸스 카페 : 코디얼 글라스에 바스푼으로 띄우기 - 그레나딘시럽 1/3part, 크렘드민트(초록색) 1/3part, 브랜디 1/3part.

화이트 와인 글라스

  1. 키르 : 화이트와인 글라스에 화이트와인 3oz , 카시스 1/2oz - 레몬껍질 꼬아서 장식.
